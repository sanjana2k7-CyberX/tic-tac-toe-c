# Tic Tac Toe in C

# Overview
This project is a console-based Tic Tac Toe game developed in C. 
The player competes against the computer on a 3Ã—3 grid. 
The program automatically detects win and draw conditions.

# Features
- Player vs Computer gameplay
- Randomized computer moves
- Win detection (rows, columns, diagonals)
- Draw detection
- Input validation
- Clear board display in terminal

# Technologies Used
- C Programming Language
- Standard Libraries:
  - stdio.h
  - stdlib.h
  - time.h

# Concepts Demonstrated
- 2D Arrays
- Functions
- Game Logic Implementation
- Random Number Generation
- Loops and Conditional Statements
